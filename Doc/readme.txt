【*】 程序简介 
-工程名称：TIM—基本定时器定时
-实验平台: 野火STM32 霸道 开发板
-MDK版本：5.16
-ST固件库版本：3.5.0

【 ！】功能简介：
使用STM32的基本定时器定时，LED每1000ms闪烁一次(亮1000ms，灭1000ms)。
学习目的：学会使用基本定时器，注意区分三种不同种类的定时器。

【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

LED：
LED灯的三个阴极接到STM32三个引脚上，LED低电平亮。

	LED_G<--->PB0
	LED_B<--->PB1
                LED_R<--->PB5
